# Codex 初音未来主题 Skill

这是 Codex Miku Theme v4 的可分发 Skill 包，包含：

- 高饱和青蓝、粉紫、雾白玻璃的全画布主题；
- 主视觉人物、侧栏纹理、拍立得与完整背景；
- 与主题配套的初音未来动态宠物；
- 精确版本检查、原始 ASAR 备份、原子安装和一键恢复。

## 安装 Skill

把 `codex-miku-theme.skill` 交给 Codex 安装，或将解压后的 `codex-miku-theme` 文件夹放入 `~/.agents/skills/`。然后对 Codex 说：

> 帮我安装初音未来主题和配套宠物。

Codex 会先做兼容性检查，再排队安装。按一次 `Command + Q` 完全退出 Codex，安装完成后应用会自动重新打开。

## 当前兼容版本

macOS Codex Desktop `26.707.72221`，构建号 `5307`。

新版本 Codex 的界面资源和 ASAR 入口可能变化，因此本包遇到未适配版本会主动拒绝安装，避免破坏应用或用旧备份覆盖更新。

## 恢复原版

对 Codex 说：

> 恢复 Codex 原版主题。

或者双击 `scripts/restore-after-quit.command`，再按 `Command + Q` 完全退出 Codex。

## 说明

这是粉丝向视觉主题。主题中涉及的角色、名称和标识权利归各自权利人所有；请仅在你有权使用相关素材的范围内安装和分发。
