正文：

```text
还在用默认灰白的 Codex？可以换了。

HeiGe Codex Skin Studio 是一个开源的 Codex Desktop 换肤工具：本机 CDP 注入，不改官方安装包，侧栏、输入框、建议卡都还是原生控件。

一张图片就是一套主题。10 套内置预设即点即换，上传自己的图自动取色，还能把 Skill 丢给 Codex，让它自己生图自己做皮肤。

仓库里带了 8 套现成的生图提示词，复制就能用。做出好看的主题，欢迎去晒图区贴一张。

GitHub：https://github.com/HeiGeAi/heige-codex-skin-studio
```
