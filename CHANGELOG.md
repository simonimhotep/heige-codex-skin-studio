# 更新日志

## 5.1.0 - 2026-07-17

### 新功能

- 全新「极光画廊」主题中心，支持主题缩略图预览、原生界面恢复和自定义图片入口。
- 主题中心与左侧菜单采用统一的玻璃质感和交互层级。

### 性能优化

- 主题切换改为先完成当前界面渲染，再在后台持久化状态，显著缩短可感知等待时间。
- 健康巡检不再反复进入耐久锁，压缩生产锁链并降低长期运行时的 CPU 占用。

### 修复

- 修复助手消息区域出现锐利白边、丢失半透明效果的问题。
- 修复切换到原生界面后被后台巡检重新切回 Miku 主题的问题。
- 修复多个对话重复展示相同 Git 和 PR 状态图标的问题。
- 修复 macOS 首次安装状态目录、Codex 精确启动和失败阶段诊断。
- 修复 Windows 计划任务身份归一化与跨 PowerShell 兼容问题。
- 修复自定义桌宠需要重启时缺少明确提示的问题。

### 发布说明

- 安装包继续使用本机回环 CDP，不修改 Codex 的 `app.asar`、应用二进制或签名资源。
- 项目所有者已确认当前视觉素材随本版本公开发布。素材权利边界与来源记录见 `ASSET_PROVENANCE.md` 和 `NOTICE.md`。
