## 5.2.2 - 2026-07-18

### 性能优化

- 后台控制器将主题请求检测与健康巡检合并为一次 renderer 扫描，减少常驻检查开销，同时保持一秒内响应主题点击。
- 主题卡片继续采用点击即切换，不增加额外保存按钮或二次确认步骤。

### 修复

- 修复 Codex renderer 无法直连控制端点时，后台巡检漏读排队请求并将内置主题错误恢复成 Miku 的问题。
- 修复主题中心「当前主题」看板随机裁切人物的问题，各内置主题现在使用独立主体焦点，优先保留人物脸部特征。
- 自定义图片在当前主题看板中改为完整展示，避免上传图片的主要内容被横向裁掉。
- 恢复字体颜色处理说明中的 👉 和 ✅ 引导符号。

### 发布说明

- 已在当前 Mac 上真实验证 Miku 与原神主题往返点击切换，后台确认后均保持正确主题。
- 安装包继续使用本机回环 CDP，不修改 Codex 的 `app.asar`、应用二进制或签名资源。

## 5.2.1 - 2026-07-17

### 新功能

- 内置主题会随皮肤自动同步 Codex 的浅色、深色或系统外观。
- 菜单上传的自定义图片会根据画面亮度自动选择 Codex 深浅外观，旧版快捷图片也能兼容补判。
- 主题面板新增字体颜色异常处理说明，可按「左下角头像 → 设置 → 外观」手动调整浅色或深色主题。
- 发布包内继续自带可选的 `Miku Future` 动画 Pet，可独立安装，不覆盖 Codex 内置宠物。

### 性能优化

- 主题点击后立即完成当前 renderer 渲染，Codex 真机实测画面切换约 52 ms。
- renderer 无法直连本机控制端点时，持久化请求改为后台排队，主题卡片保持可操作。
- 快速连续切换主题时合并等待中的请求，仅持久化最后一次选择。

### 修复

- 修复 CSP 阻止 renderer 请求控制端点后，主题按钮长时间锁定并产生约 2 至 3 秒可感知延迟的问题。
- 修复深色与浅色图片切换后，Codex 设置中的外观没有同步变化的问题。

### 发布说明

- 安装包内含可选 `Miku Future` Pet，皮肤安装不会自动启用 Pet。
- 安装包继续使用本机回环 CDP，不修改 Codex 的 `app.asar`、应用二进制或签名资源。

## 5.1.0 - 2026-07-17

### 新功能

- 全新「极光画廊」主题中心，支持主题缩略图预览、原生界面恢复和自定义图片入口。
- 主题中心与左侧菜单采用统一的玻璃质感和交互层级。

### 性能优化

- 主题切换改为先完成当前界面渲染，再在后台持久化状态，显著缩短可感知等待时间。
- 健康巡检不再反复进入耐久锁，压缩生产锁链并降低长期运行时的 CPU 占用。

### 修复

- 修复助手消息区域出现锐利白边、丢失半透明效果的问题。
- 修复切换到原生界面后被后台巡检重新切回 Miku 主题的问题。
- 修复多个对话重复展示相同 Git 和 PR 状态图标的问题。
- 修复 macOS 首次安装状态目录、Codex 精确启动和失败阶段诊断。
- 修复 Windows 计划任务身份归一化与跨 PowerShell 兼容问题。
- 修复自定义桌宠需要重启时缺少明确提示的问题。

### 发布说明

- 安装包继续使用本机回环 CDP，不修改 Codex 的 `app.asar`、应用二进制或签名资源。
- 项目所有者已确认当前视觉素材随本版本公开发布。素材权利边界与来源记录见 `ASSET_PROVENANCE.md` 和 `NOTICE.md`。
